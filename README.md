# Pricing UI Component

A responsive and interactive pricing UI built using HTML, CSS, and JavaScript.

## 🚀 Features
- Responsive pricing cards (Basic, Standard, Premium)
- Smooth hover animations and transitions
- JavaScript-based plan selection and highlighting
- Clean and reusable UI design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
